# awesome-zap

A curated list of projects built with the **Zap** programming language.

This repository aims to collect useful, inspiring, and actively maintained Zap projects across different categories (libraries, tools, apps, and learning resources).

## What belongs here

- Open-source projects written in Zap
- Libraries and frameworks
- Developer tooling and CLI utilities
- Real-world applications and demos
- Learning resources and examples

## How to add a project

If you want to add a project, open a Pull Request and include it in the appropriate section.

### Entry format

Use this format for each item:

- `ProjectName` — short, specific description. [GitHub/Website](https://example.com)

## Contribution guidelines

To keep this list useful and trustworthy, please follow these rules:

1. **Zap-related only**  
   The project should be written in Zap (fully or in a meaningful part).

2. **Publicly accessible**  
   The project link must be public and reachable.

3. **Clear description**  
   Keep it short and concrete (one sentence is enough).

4. **No duplicates**  
   Check whether the project is already listed before adding it.

5. **Alphabetical order**  
   Keep entries sorted alphabetically within each section.

6. **Working links**  
   Ensure links point to the correct project page/repository.
